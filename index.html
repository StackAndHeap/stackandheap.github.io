Randori demo application

As a proof of concept and concrete implementation of the Randori Framework, we've created a demo Pet Store.

In this application we tried to implement all the core functionality of the framework and created our own inputcomponents, router and mock services.

We've written articles about our experiences and some best practices while using Randori, you can find these at our blog.
